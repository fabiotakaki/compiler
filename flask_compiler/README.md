# Compiler


# Steps to run

Clone the repository and using your virtualenv, run the following steps on terminal:

1. pip install -r requirements.txt
2. export FLASK_APP=flask_compiler/flask_compiler.py
3. export FLASK_DEBUG=true
4. flask run
